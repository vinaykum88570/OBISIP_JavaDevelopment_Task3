<h2> Project Name :  ATM Interface System | Java + Spring Boot + MySQL(Task 3)</h2>
<p>Excited to showcase my ATM Interface System project developed using Java, Spring Boot, Spring Data JPA, and MySQL.

This project simulates real-world ATM functionalities where users can register, log in securely using ATM ID and PIN, perform banking operations, and manage transactions through a console-based interface.</p>
<h2>🔹 Key Features:</h2>
<ul>
<li> ATM User Registration & Secure Login</li>
<li>Deposit & Withdraw Money</li>
<li>Money Transfer Between Accounts</li>
<li>Transaction History Management</li>
<li>Real-time Balance Updates</li>
<li>Credit & Debit Transaction Tracking</li>
<li>MySQL Database Integration using JPA/Hibernate</li>
<li>Layered Architecture (Service, Repository, Entity, Operation Layers)</li>
</ul>
<h2>🔹 Technologies Used:</h2>
<ul>
  <li>☕ Java</li>
<li>🌱 Spring Boot</li>
<li>🗄️ MySQL</li>
<li>📦 Spring Data JPA / Hibernate</li>
<li>🔧 Maven</li>
<li>💻 Eclipse / VS Code</li>
</ul>

<h2>🔹 My Contribution:</h2>
<p>I designed and developed the entire backend system, implemented secure login authentication, managed banking operations such as deposit, withdrawal, and money transfer, and integrated MySQL database connectivity using Spring Boot and JPA.

One of the major functionalities I implemented was maintaining separate sender and receiver transaction records for transfer history, along with automatic balance updates after each transaction.

This project helped me strengthen my understanding of Java backend development, database management, Spring Boot architecture, CRUD operations, and real-world banking system workflows.</p>
